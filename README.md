# Orbital-Angular-Momentum
涡旋电磁波相关仿真

本项目将分为以下几方面：

1. SourceFile 当前已经初步完成代码(后续会进一步完善)
2. WaitToArchive 等待归档的代码（主要为各论文的仿真）
3. WaitToCompleted 仍存在问题的代码

SourceFile 内部结构的初步设想：

1. Algorithm          算法相关代码
2. Domes              代码示例
3. Documentation      文档说明
4. Geometry           坐标相关代码
5. Plot               绘图相关代码
6. Signal             信号相关代码
